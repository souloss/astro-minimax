## Welcome to the Demo Blog

This blog is built using the **@astro-minimax/core** and **@astro-minimax/viz** npm packages from the astro-minimax monorepo.

### What's Included

- Full blog layout and navigation from `@astro-minimax/core`
- Visualization components from `@astro-minimax/viz`
- Mermaid diagrams, Markmap mind maps
- Dark mode support
- Syntax-highlighted code blocks

### Code Example

```typescript
// Using @astro-minimax packages
import Layout from "@astro-minimax/core/layouts/Layout.astro";
import Mermaid from "@astro-minimax/viz/components/Mermaid.astro";
```

This demonstrates that the packages are correctly installed and functional.
