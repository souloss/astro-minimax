## What is Markmap?

Markmap is a tool that converts Markdown into interactive mind maps, perfect for presenting hierarchical content.

## Basic Usage

```markmap
# Central Topic

## Branch 1
### Sub-branch 1-1
### Sub-branch 1-2
#### Deeper level

## Branch 2
### Sub-branch 2-1
### Sub-branch 2-2

## Branch 3
### Sub-branch 3-1
```
